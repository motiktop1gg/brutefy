# ⚡️ brutefy ⚡️

> A lightweight, fast, and multi-threaded Wi-Fi handshake and brute-force tool written in Python. 

---

## 🚀 Features

* **🎯 Targeted Attacks:** Easily target specific SSIDs or automated network scanning.
* **🛡 Smart Logging:** Clean console output with real-time stats and cracked credentials saving.
* **🧩 Clean Architecture:** No bloated code, dependency-light, and easy to modify.

---

## 🛠 Installation

### 1. Clone the repository
git clone https://github.com/motiktop1gg/brutefy.git
cd brutefy

### 2. Install Dependencies
pip install -r requirements.txt

---

## 💻 Usage

To run the tool, pass your wordlist and target parameters:

python main.py 5

### Options:
* ct — Connecting time to WiFi.

> **💡 OpSec Tip:** For better anonymity and OPSEC, it is highly recommended to route your traffic through a trusted **VPN** and configure **MAC address randomization** (e.g., using `macchanger`) before launching any attacks.
